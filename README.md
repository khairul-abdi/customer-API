# CRUD Customer APP

### In this App, we need 4 dependencies below:

- Express
- Body Parser
- MySQL (mysql driver for node.js)
- Bootstrap

### Quick Start

```bash
# Install dependencies
$ npm install

# Database

$ node_crud.sql

# Start App

$ node server.js

App Run On: http://localhost:3000

```

### Test the APIs

```bash
Server is running on port 3000.
Successfully connected to the database.
```
